    # Monarch Rules

    Monarch rules are a set of constraints on our knowledge graph, currently represented in cypher but with future plans to leverage sparql and shex.

    These rules are queried and reports generated after every data load with jenkins.

    Rules and cypher queries are listed in the qc yaml configuration

    Or alternatively, create a ticket requesting a new rule in the qc repo

    This report is based on the versions:
    Production build: https://archive.monarchinitiative.org/202011
    Beta build: https://archive.monarchinitiative.org/202012
